# MyApplication Trackr

A simple and intuitive web application for tracking and managing job or opportunity applications in one place.


---

## 📖 Overview

Application Trackr helps users stay organized during their job search by providing a clean interface to manage multiple applications efficiently.

Instead of juggling spreadsheets or notes, users can track everything in one place — from application status to company details.

---

## ✨ Features

- 📋 Add new applications with relevant details
- ✏️ Edit and update application information
- ❌ Delete applications
- 📊 Track application status (e.g. Applied, Interview, Rejected, Accepted)
- 💾 Data stored locally in the browser (no backend required)
- 📱 Fully responsive design (works on mobile & desktop)

---

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling & layout
- **JavaScript (Vanilla JS)** – Functionality & interactivity
- **LocalStorage API** – Data persistence
- **GitHub Pages** – Deployment

---

## 🧠 How It Works

1. Users input application details (company, role, status, etc.)
2. Data is saved in the browser using `localStorage`
3. Applications are displayed dynamically on the UI
4. Users can update or delete entries anytime
